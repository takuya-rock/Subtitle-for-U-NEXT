# Subtitle for U-NEXT

**U-NEXTの動画プレイヤー上に、外部の英語字幕（SRT形式）を表示・同期させるためのChrome拡張機能です。** U-NEXT標準の字幕では物足りない英語学習者向けに、ネットからの字幕検索と、細かな同期調整機能を提供します。

---

## 🚀 主な機能

- **オンライン字幕検索**: `OpenSubtitles.com` APIと連携し、作品名から直接英語字幕を検索・ダウンロード。
- **外部SRT読み込み**: ローカルにあるSRTファイルを動画に重ねて表示。
- **自動同期マネージャー**: 
    - 字幕リストから特定のセリフを選んで、再生中のシーンに一発同期。
    - 前後のセリフへの移動も簡単。
- **表示カスタマイズ**: 字幕のサイズ（px）、表示位置（%）、同期オフセット（秒）を、動画を見ながらリアルタイムに調整可能。
- **自動保存**: 動画ごとに設定した位置、サイズ、同期オフセットを自動的に記憶し、次回の視聴時に復元。

## 🛠 導入手順

本ツールを利用するには、ご自身で取得したOpenSubtitlesのAPIキーを拡張機能の設定画面に入力する必要があります。

### 1. ソースコードの準備
GitHubからリポジトリをクローンするか、[Releases](https://github.com/takuya-rock/Subtitle-for-U-NEXT/releases)から最新のソースコード（ZIP）をダウンロードして展開してください。

### 2. Chromeへのインストール
1. Chromeのアドレスバーに `chrome://extensions/` を入力。
2. 右上の「デベロッパーモード」をONにする。
3. 「パッケージ化されていない拡張機能を読み込む」を選択し、展開したフォルダを選択。

### 3. APIキーの取得と設定
1. [OpenSubtitles.com](https://www.opensubtitles.com/) でアカウントを作成。
2. [API Consumers](https://www.opensubtitles.com/ja/consumers) でアプリ（Consumer）を登録し、`Api-Key` を取得します。
   - ※個人利用（1日100件無料）の場合は、登録時に **"Under dev"** にチェックを入れてください。
3. U-NEXTのビデオ再生画面を開きます。
4. 本ツールの検索バー右側にある **鍵アイコン（🔑）** をクリックして設定パネルを開きます。
5. 取得した `Api-Key` と `User-Agent (App Name)` を入力し、「設定を保存」をクリックしてください。

## 👨‍💻 技術的なポイント

- **CORS回避**: `declarativeNetRequest` を使用し、ブラウザのリクエストヘッダーから `Origin` を削除、`User-Agent` を動的に書き換えることでAPIサーバーとの403エラーを回避しています。
- **Service Worker連携**: 通信処理を `background.js` に集約。設定されたAPIキーを `chrome.storage` から読み込み、安全かつ安定した通信を実現しています。
- **シネマティックUI**: U-NEXTのプレイヤー構造を監視し、設定パネルや字幕インジケーターを動的に挿入。再生中の邪魔にならない透過デザインを採用しています。
