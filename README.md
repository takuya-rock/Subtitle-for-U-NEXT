# Subtitle for U-NEXT

**U-NEXTの動画プレイヤー上に、外部の英語字幕（SRT形式）を表示・同期させるためのChrome拡張機能です。**  
U-NEXT標準の字幕では物足りない英語学習者向けに、ネットからの字幕検索と、細かな同期調整機能を提供します。

---

## 🚀 主な機能

- **外部SRT読み込み**: ローカルにあるSRTファイルを動画に重ねて表示。
- **オンライン字幕検索**: `OpenSubtitles.com` APIと連携し、作品名から直接英語字幕を検索・ダウンロード。
- **自動同期マネージャー**: 
    - 字幕リストから特定のセリフを選んで、再生中のシーンに一発同期。
    - 前後のセリフへの移動も簡単。
- **表示カスタマイズ**: 字幕のサイズ（px）、表示位置（%）、同期オフセット（秒）を、動画を見ながらリアルタイムに調整可能。
- **自動保存**: 設定した位置やサイズ、同期オフセットは、次回の視聴時にも自動で復元される。

## 🛠 導入手順

この拡張機能は、自分でAPIキーを設定して使用する開発者・学習者向けのツールです。

### 1. リポジトリのクローン
```bash
git clone https://github.com/takuya-rock/Subtitle-for-U-NEXT.git
```

### 2. 設定ファイルの作成
1. プロジェクトルートにある `config.example.js` をコピーして、`config.js` という名前にリネームする。
   - ※ `config.js` は `.gitignore` に登録されているため、Git管理（Push）からは除外される。
2. `config.js` を開き、以下の情報を記入する。
   ```javascript
   const API_KEY = "あなたのAPIキー";
   const USER_AGENT = "登録したApp Name";
   ```

### 3. APIキーの取得方法
1. [OpenSubtitles.com](https://www.opensubtitles.com/) でアカウントを作成。
2. [API Consumers](https://www.opensubtitles.com/ja/consumers) でアプリ（Consumer）を登録。
   - 開発用（1日100件無料）として使うには、登録時に **"Under dev"** にチェックを入れること。
3. 発行された `Api-Key` と、登録した `Name` (User-Agent) を `config.js` に貼り付ける。

### 4. Chromeへのインストール
1. Chromeのアドレスバーに `chrome://extensions/` を入力。
2. 右上の「デベロッパーモード」をONにする。
3. 「パッケージ化されていない拡張機能を読み込む」を選択し、本リポジトリのフォルダを選択。

## 👨‍💻 技術的なポイント

- **CORS回避**: `declarativeNetRequest` を使用し、ブラウザのリクエストヘッダーから `Origin` を削除、`User-Agent` を偽装することでAPIサーバーとの403エラーを回避している。
- **Service Worker連携**: 通信処理（Fetch）を `background.js` に集約し、コンテンツスクリプトとメッセージングを行うことで、安全で堅牢な通信を実現。
- **シネマティックUI**: 字幕インジケーターや設定パネルを、動画プレイヤーのDOM構造に動的に挿入。再生中の邪魔にならない透過デザインを採用。
